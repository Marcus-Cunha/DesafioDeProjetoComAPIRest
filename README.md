# Desafio de Projeto Desenvolvimento de testes unitários para validar uma API REST de gerenciamento estoques de cerveja

Testando uma API REST para o gerenciamento de estoques de cerveja desenvolvendo testes unitários para validar o nosso sistema de gerenciamento de estoques de cerveja.



Para executar o projeto no terminal, digite o seguinte comando:

```shell script
mvn spring-boot:run 
```

Para executar a suíte de testes desenvolvida durante a live coding, basta executar o seguinte comando:

```shell script
mvn clean test
```

Após executar o comando acima, basta apenas abrir o seguinte endereço e visualizar a execução do projeto:

```
http://localhost:8080/api/v1/beers
```
